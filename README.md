# Databricks Sales Lakehouse (Bootcamp Learning Project)

This is my personal learning project based on **[Databricks Bootcamp 2026](https://github.com/DataWithBaraa/databricks_bootcamp_2026)** by **Data With Baraa**.

## Project Overview

Following the bootcamp roadmap and video lessons, I built this Lakehouse project step by step across the **Bronze / Silver / Gold** layers.

All notebooks and code in this repository were written by me after completing each lesson, and later refined with additional improvements.

## Data

For the original dataset files, please refer to:

## Dashboard Notes

Because I use macOS, I could not directly connect to Power BI in my local setup. Instead, I created simple dashboards directly in Databricks and saved screenshots in the `dashboard/` folder.

The main point here is to demonstrate that I can use Databricks dashboard features in practice. Since this is a bootcamp-scale dataset, deep dashboard analysis is not the primary focus of this repository.
